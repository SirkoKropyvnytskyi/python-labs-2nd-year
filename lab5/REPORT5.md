3. Код програми
```python

import os

file_name = 'data.txt'
line_to_add = "cool new line that is very (not) important and definitely not the screams of the damned"

if not os.path.exists(file_name):
    with open(file_name, 'w', encoding='utf-8') as f:
        f.write("test file text.\n")

with open(file_name, 'a', encoding='utf-8') as f:
    f.write(f"added line: {line_to_add}\n")

print(f"New line added. Please check file {file_name}")
```

4. Висновок

1.import os (підключення стандартного модуля Python для взаємодії з операційною системою, що дозволяє виконувати перевірку наявності файлів на диску).

2.Оголошення змінних file_name та line_to_add (визначення назви текстового документа та змісту нового рядка для централізованого керування даними в коді).

3.if not os.path.exists(file_name): (логічна перевірка існування файлу, яка повертає «істину», якщо файл не знайдено, що дозволяє уникнути помилок при спробі доступу до неіснуючого ресурсу).

4.Режим 'w' у першому блоці with (режим запису «write», який створює новий файл або повністю перезаписує існуючий, видаляючи всі старі дані).

5.Режим 'a' у другому блоці with (режим дозапису «append», за якого нові дані додаються виключно в кінець файлу, що дозволяє зберегти попередню інформацію цілісною).

6.encoding='utf-8' (параметр кодування тексту, що забезпечує правильне розпізнавання українських літер та символів кирилиці, запобігаючи появі помилок при зчитуванні).

7.Конструкція with ... as f: (менеджер контексту, який автоматично закриває файл після завершення операцій, звільняючи системні ресурси та гарантуючи збереження змін).

8.Метод f.write() (вбудована функція для безпосереднього внесення текстової інформації у відкритий файл).

9.Використання f-string f"added line: {line_to_add}\n" (метод форматування рядків, що дозволяє динамічно вставляти значення змінних у текст разом із символом перенесення рядка \n).

Опановано принципи роботи з текстовими файлами в Python, керування файловими ресурсами через режими 'w' (write, запис з перезаписом) та 'a' (дозапис). На основі розробленого коду я закріпив навичку використання конструкції with, яка виступає контекстним менеджером, забезпечує автоматичне закриття файлів та збереження змін, звільнення ресурсів та є ключовим для безпечного управління потоками даних і запобігання втраті даних. Використана функція os.path.exists(), яка перевіряє існування текстового файлу. Для коректної обробки символів, зокрема в контексті можливого використання кирилиці або спеціальних знаків, у аргументах відкриття вказано параметр encoding='utf-8'. Застосували write() для внесення інформації, який виконує функцію прямого передавання символьного рядка у відкритий файловий потік для його фізичного збереження на диску. Застосували f-рядкок — це інструмент строкової інтерполяції, який дозволяє на етапі виконання програми «склеїти» статичний текст (маркер «added line:») із динамічним вмістом змінної line_to_add у єдиний текстовий об'єкт. Підсумовуючи я опанував основні режими відкриття файлів (читання, запис з перезаписом, дозапис, читання та запис одночасно), методи роботи з файлами, принципи збереження та закриття файлів, основи обробки рядків при роботі з файлами, відкриття текстових файлів для читання та запису, зчитування даних з файлів і запис їх у новий файл, перетворення та обробку тексту файлів, додавання нових даних до існуючого файлу і перезапис, перевірку наявності файлу перед роботою з цим файлом, а також набув навичок розробки програм для обробки текстових файлів, роботи з потоками даних та використання конструкції with для безпечного управління файлами.
