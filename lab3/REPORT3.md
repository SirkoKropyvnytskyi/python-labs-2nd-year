3. Код програми
```python

shop = ["health potion", "legendary sword", "dark armor", "mana potion"]

print("List before sorting:", shop)

shop.sort()

print("List after sorting:", shop)
```

4. Висновок

Я ознайомився зі складними типами даних у мові Python — списками та словниками. Cтворив програму для сортування списку за зростанням.

Список shop використано як впорядковану змінювану послідовність для зберігання текстових назв товарів.

Сортування елементів за зростанням (в алфавітному порядку для рядкових даних) реалізовано за допомогою методу sort(). Аналогічно цей метод може бути використаний для роботи з іншими типами даних, наприклад, для впорядкування цілих чисел або чисел із плаваючою комою за їх числовим значенням.

Метод sort() було обрано тому, що він дозволяє ефективно змінювати порядок елементів безпосередньо у початковому списку.
