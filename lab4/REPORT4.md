3. Код програми
```python

def calculate_square_area(side):
    """Функція повертає площу квадрата за заданою стороною"""
    area = side ** 2
    return area

def main():
    s = float(input("Введіть довжину сторони квадрата: "))
    
    square_area = calculate_square_area(s)
    
    print(f"Площа квадрата зі стороною {s} дорівнює {square_area}")
 
if __name__ == "__main__":
    main()
```

4. Висновок

1) Функція calculate_square_area приймає один параметр side (сторона квадрата), обчислює площу шляхом піднесення до квадрата та повертає результат за допомогою оператора return.
2) Основна програма (main):
* зчитує значення сторони від користувача за допомогою input() та перетворює його на число з плаваючою крапкою (float);
* викликає функцію calculate_square_area для обчислення площі;
* виводить отриманий результат на екран у зручному форматі.
3) Блок if __name__ == "__main__": дозволяє запускати логіку програми лише при безпосередньому виконанні файлу, що забезпечує модульність коду.
